# Building a Quantum Simulator
Hi! I am building Quantum Qubit Simulator due to my interest in Quantum Programming. I have made program from scratch want to learn how math actually work instead of premade libraries.

## What I have built so far:

### Step 1: My Classical Simulator (Quantum_Simulator_step1.py)
The program takes a percentage like (0.7 for 70%) and use random number generator to find out if qubit is 1 or 0.


### Step 2: My Quantum Amplitude Simulator (Quantum_Amplitude_step2.py)
My first code was beginner which do not use math that's why I use Quantum Amplitude.
Because quantum math squares the amplitude to get percentage .To get the chance of 70 we have to pass sqrt of o.7 like this (math.sqrt(0.7))
